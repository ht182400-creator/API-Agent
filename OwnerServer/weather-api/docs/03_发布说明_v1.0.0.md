## 1. 版本概述

Weather API v1.0.0 是天气 API 产品的首个正式版本，提供基础的天气数据查询功能。

### 1.1 主要功能

| 功能 | 描述 | 状态 |
|------|------|------|
| 实时天气查询 | 获取当前城市天气数据 | ✅ 已上线 |
| 天气预报查询 | 获取未来7天天气预报 | ✅ 已上线 |
| 空气质量查询 | 获取 AQI 及污染物浓度 | ✅ 已上线 |
| 天气预警查询 | 获取灾害天气预警信息 | ✅ 已上线 |

### 1.2 支持的城市

| 范围 | 城市数量 | 示例 |
|------|----------|------|
| 中国大陆 | 366+ | 北京、上海、广州、深圳 |
| 港澳台 | 3+ | 香港、澳门、台北 |
| 国际主要城市 | 100+ | 纽约、东京、伦敦 |

## 8. 升级指南

### 8.1 从测试版升级

如果您正在使用测试版 API Key，请按以下步骤升级：

1. 登录 [开发者控制台](https://console.platform.com)
2. 进入「天气 API」→「应用管理」
3. 点击「升级到正式版」
4. 选择适合的套餐
5. 完成支付后自动生效

### 8.2 兼容性说明

- API v1.0.0 完全兼容测试版接口
- 测试版 Key 可直接用于正式环境
- 响应格式保持不变
