## 三、测试执行

### 3.1 前端测试

```powershell
# 进入前端目录
cd d:\Work_Area\AI\API-Agent\api-platform\web

# 运行所有 E2E 测试
npx playwright test

# 运行特定测试文件
npx playwright test e2e/auth.spec.ts
npx playwright test e2e/keys.spec.ts

# 运行错误处理测试
npx playwright test -g "错误"
```

### 3.2 后端测试

```powershell
# 进入后端目录
cd d:\Work_Area\AI\API-Agent\api-platform

# 运行所有测试
python -m pytest tests/ -v

# 运行特定测试
python -m pytest tests/test_quota_api.py -v
python -m pytest tests/test_billing.py -v
```
