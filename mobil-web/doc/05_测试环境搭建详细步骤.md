### 2.4 使用自定义设备配置

**方法1：手动添加（推荐）**

```
1. 打开Chrome DevTools
2. 点击设备图标 📱
3. 设备下拉列表 → "Edit..."
4. 点击 "Add custom device..."
5. 从配置文件中复制参数并填写
6. 点击 "Add"
7. 重复步骤4-6，添加所有设备
```

**方法2：使用Playwright脚本（高级）**

```typescript
/**
 * 自动添加自定义设备到Chrome
 * 注意：Chrome不提供直接导入设备的API
 * 此脚本用于生成配置说明
 */

const devices = require('./chinese-devices.json');

console.log('请在Chrome DevTools中手动添加以下设备：\n');

devices.forEach(device => {
  console.log(`设备名称：${device.name}`);
  console.log(`宽度：${device.width}px`);
  console.log(`高度：${device.height}px`);
  console.log(`设备像素比：${device.deviceScaleFactor}`);
  console.log(`用户代理：${device.userAgent}`);
  console.log('---');
});
```

## 3. 国产手机测试环境

### 3.1 华为手机测试环境

**测试设备**：华为Mate 60 Pro（HarmonyOS 4.0）

#### 3.1.1 环境搭建步骤

```
步骤1：开启开发者模式
  1. 打开"设置"
  2. 滑动到最下方，点击"关于手机"
  3. 连续点击"版本号"7次
  4. 输入锁屏密码
  5. 提示"您已进入开发者模式"

步骤2：开启USB调试
  1. 返回"设置"
  2. 点击"系统和更新"
  3. 点击"开发者选项"
  4. 开启"USB调试"
  5. 开启"仅充电模式下允许ADB调试"

步骤3：连接电脑
  1. 使用USB线连接手机和电脑
  2. 在手机上，下拉通知栏
  3. 点击"正在通过USB充电"
  4. 选择"传输文件"
  5. 在弹出的"允许USB调试？"对话框中，点击"允许"
```

#### 3.1.2 Chrome远程调试

```bash
# 步骤1：查看连接的设备
adb devices

# 应该看到类似输出：
# List of devices attached
# ACL0219C10019123    device

# 步骤2：启动Chrome并启用远程调试
# 在手机上打开Chrome浏览器
# 访问 http://电脑IP:3000

# 步骤3：在电脑Chrome中调试
# 1. 打开Chrome
# 2. 地址栏输入：chrome://inspect
# 3. 找到你的设备
# 4. 点击"Inspect"
```

#### 3.1.3 微信浏览器测试

```
步骤1：开启微信调试模式
  1. 在微信中访问：http://debugxweb.qq.com/?inspect
  2. 开启"启用Inspect调试功能"
  3. 重启微信

步骤2：远程调试
  1. 在微信中访问你的页面
  2. 在电脑Chrome中打开：chrome://inspect
  3. 找到微信进程
  4. 点击"Inspect"
```

## 4. 海外手机测试环境

### 4.1 iPhone测试环境

**前提条件**：
- Mac电脑（必需，因为需要Safari）
- iPhone手机
- USB数据线

#### 4.1.1 环境搭建步骤

```
步骤1：iPhone开启Web检查器
  1. 打开"设置"
  2. 找到"Safari"
  3. 滑动到最下方
  4. 点击"高级"
  5. 开启"Web检查器"

步骤2：Mac开启Safari开发菜单
  1. 打开Safari
  2. 点击顶部菜单"Safari" → "偏好设置"
  3. 点击"高级"
  4. 勾选"在菜单栏中显示"开发"菜单"

步骤3：连接iPhone
  1. 使用USB线连接iPhone和Mac
  2. 如果是第一次连接，iPhone会提示"信任此电脑？"
  3. 点击"信任"
  4. 输入iPhone锁屏密码

步骤4：开始调试
  1. 在iPhone的Safari中访问你的页面
  2. 在Mac的Safari中
  3. 点击顶部菜单"开发" → 选择你的iPhone → 选择页面
  4. Web检查器会打开
```

#### 4.1.2 使用Chrome调试iPhone（有限支持）

```bash
# 注意：Chrome无法直接调试iOS Safari
# 但可以使用第三方工具

# 方案1：使用Safari Technology Preview
# 下载：https://developer.apple.com/safari/technology-preview/

# 方案2：使用weinre（远程调试工具）
npm install -g weinre

# 启动weinre服务器
weinre --boundHost -all-

# 在iPhone页面中注入脚本
<script src="http://电脑IP:8080/target/target-script-min.js#anonymous"></script>

# 在电脑浏览器中打开
http://localhost:8080/client/#anonymous
```
