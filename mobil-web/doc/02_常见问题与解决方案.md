## 1. 布局问题

### 1.1 问题：移动端出现横向滚动条

**现象**：
```
┌────────────────────────────────────────┐
│  页面内容                            │← 出现横向滚动条
└────────────────────────────────────────┘
```

**原因**：
- 元素宽度超出视口
- 使用了固定宽度
- padding/margin 导致溢出

**解决方案**：

```css
/* 方案1：使用max-width替代width */
.container {
  max-width: 100%;  /* ✅ 正确 */
  /* width: 100%; */ /* ❌ 可能导致溢出 */
}

/* 方案2：盒模型设置 */
* {
  box-sizing: border-box;  /* ✅ 确保padding不增加宽度 */
}

/* 方案3：防止内容溢出 */
img, table, pre {
  max-width: 100%;
  height: auto;
  overflow-x: auto;
}

/* 方案4：视口单位 */
.full-width {
  width: 100vw;  /* ❌ 可能导致滚动条 */
  width: 100%;    /* ✅ 正确 */
}
```

**检查清单**：
- [ ] 所有元素使用 `max-width: 100%`
- [ ] 盒模型设置为 `border-box`
- [ ] 没有固定宽度（如 `width: 500px`）
- [ ] 表格/图片等可滚动元素添加 `overflow-x: auto`

### 1.2 问题：虚拟键盘弹起导致布局错乱

**现象**：
```
┌────────────────────────────────────────┐
│  输入框                              │
│  ┌──────────────────┐               │
│  │                  │               │
│  └──────────────────┘               │
│                                      │
│  【虚拟键盘弹起】                      │
│  ┌──────────────────┐               │
│  │                  │               │
│  └──────────────────┘               │
│                                      │
│  按钮被键盘遮挡 ← 问题                │
└────────────────────────────────────────┘
```

**原因**：
- 使用 `height: 100vh` 在移动端不准确
- 虚拟键盘改变视口高度

**解决方案**：

```css
/* 方案1：使用min-height替代height */
.layout {
  min-height: 100vh;  /* ✅ 正确 */
  /* height: 100vh; */  /* ❌ 可能导致溢出 */
}

/* 方案2：使用flex布局 */
.layout {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

.content {
  flex: 1;
}

/* 方案3：JavaScript动态计算 */
const useViewportHeight = () => {
  useEffect(() => {
    const setVH = () => {
      const vh = window.innerHeight * 0.01;
      document.documentElement.style.setProperty('--vh', `${vh}px`);
    };
    
    setVH();
    window.addEventListener('resize', setVH);
    
    return () => window.removeEventListener('resize', setVH);
  }, []);
};

/* 使用 */
.layout {
  height: calc(var(--vh, 1vh) * 100);
}
```

## 2. 触摸交互问题

### 2.1 问题：点击延迟300ms

**现象**：
- 点击按钮后，300ms后才触发事件
- 用户体验差

**原因**：
- 浏览器等待判断是否双击

**解决方案**：

```typescript
/**
 * 方案1：使用touch事件替代click
 */
const useFastClick = (ref: RefObject<HTMLElement>) => {
  useEffect(() => {
    const element = ref.current;
    if (!element) return;
    
    const handleTouchEnd = (e: TouchEvent) => {
      e.preventDefault();
      element.click();
    };
    
    element.addEventListener('touchend', handleTouchEnd, { passive: false });
    
    return () => {
      element.removeEventListener('touchend', handleTouchEnd);
    };
  }, [ref]);
};

/**
 * 方案2：CSS属性
 */
.button {
  touch-action: manipulation;  /* 禁用双击缩放 */
}

/**
 * 方案3：meta标签
 */
/* index.html */
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no" />
```

### 2.2 问题：滚动穿透

**现象**：
- 在弹窗/抽屉中滚动时，底层页面也跟着滚动

**解决方案**：

```typescript
/**
 * 防止滚动穿透Hook
 */
export const usePreventScrollLeak = (isOpen: boolean) => {
  useEffect(() => {
    if (isOpen) {
      // 记录当前滚动位置
      const scrollY = window.scrollY;
      
      // 固定body
      document.body.style.position = 'fixed';
      document.body.style.width = '100%';
      document.body.style.top = `-${scrollY}px`;
      
      return () => {
        // 恢复body
        document.body.style.position = '';
        document.body.style.width = '';
        document.body.style.top = '';
        
        // 恢复滚动位置
        window.scrollTo(0, scrollY);
      };
    }
  }, [isOpen]);
};

/**
 * 使用方法
 */
const Drawer: React.FC = () => {
  const [open, setOpen] = useState(false);
  
  usePreventScrollLeak(open);
  
  return (
    <div>
      {open && (
        <div className="drawer">
          {/* 抽屉内容 */}
        </div>
      )}
    </div>
  );
};
```

### 2.3 问题：滑动时误触按钮

**现象**：
- 用户在页面上滑动时，误触了按钮

**解决方案**：

```typescript
/**
 * 防止滑动误触Hook
 */
export const usePreventAccidentalClick = (
  ref: RefObject<HTMLElement>
) => {
  const startX = useRef(0);
  const startY = useRef(0);
  
  useEffect(() => {
    const element = ref.current;
    if (!element) return;
    
    const handleTouchStart = (e: TouchEvent) => {
      startX.current = e.touches[0].clientX;
      startY.current = e.touches[0].clientY;
    };
    
    const handleTouchEnd = (e: TouchEvent) => {
      const diffX = Math.abs(e.changedTouches[0].clientX - startX.current);
      const diffY = Math.abs(e.changedTouches[0].clientY - startY.current);
      
      // 如果移动距离超过10px，认为是滑动，阻止点击
      if (diffX > 10 || diffY > 10) {
        e.preventDefault();
      }
    };
    
    element.addEventListener('touchstart', handleTouchStart, { passive: true });
    element.addEventListener('touchend', handleTouchEnd, { passive: false });
    
    return () => {
      element.removeEventListener('touchstart', handleTouchStart);
      element.removeEventListener('touchend', handleTouchEnd);
    };
  }, [ref]);
};
```

```tsx
      itemCount={data.length}

      {Row}
    </List>
  );

/**
 * 方案2：分页加载
 */
const PaginatedList: React.FC = () => {
  const [page, setPage] = useState(1);
  const [data, setData] = useState<any[]>([]);
  const [hasMore, setHasMore] = useState(true);
  
  const loadMore = async () => {
    const newData = await api.getList({ page, pageSize: 20 });
    setData(prev => [...prev, ...newData]);
    setHasMore(newData.length === 20);
    setPage(prev => prev + 1);
  };

        <div key={item.id}>{item.name}</div>

    </div>

/**
 * 方案3：无限滚动
 */
const useInfiniteScroll = (
  loadMore: () => Promise<void>,
  hasMore: boolean
) => {
  const observerRef = useRef<IntersectionObserver>();
  
  const sentinelRef = useCallback((node: HTMLDivElement | null) => {
    if (observerRef.current) {
      observerRef.current.disconnect();
    }
    
    observerRef.current = new IntersectionObserver(entries => {
      if (entries[0].isIntersecting && hasMore) {
        loadMore();
      }
    });
    
    if (node) {
      observerRef.current.observe(node);
    }
  }, [hasMore, loadMore]);
  
  return sentinelRef;
};
```

### 3.2 问题：图片加载慢

**解决方案**：

```tsx
/**
 * 方案1：懒加载
 */
const LazyImage: React.FC<{ src: string; alt: string }> = ({ src, alt }) => {
  const [loaded, setLoaded] = useState(false);
  const imgRef = useRef<HTMLImageElement>(null);
  
  useEffect(() => {
    const observer = new IntersectionObserver(entries => {
      entries.forEach(entry => {
        if (entry.isIntersecting) {
          const img = entry.target as HTMLImageElement;
          img.src = src;
          observer.unobserve(img);
        }
      });
    });
    
    if (imgRef.current) {
      observer.observe(imgRef.current);
    }
    
    return () => observer.disconnect();
  }, [src]);
  
  return (
    <img
      ref={imgRef}
      alt={alt}
      className={loaded ? 'loaded' : 'loading'}
      onLoad={() => setLoaded(true)}
    />
  );
};

/**
 * 方案2：图片压缩
 */
// 使用工具压缩图片
// - TinyPNG: https://tinypng.com/
// - ImageOptim: https://imageoptim.com/

/**
 * 方案3：WebP格式
 */
<picture>
  <source srcSet="image.webp" type="image/webp" />
  <img src="image.jpg" alt="备用图片" />
</picture>
```

## 4. 兼容性问题

### 4.1 问题：iOS日期解析失败

**现象**：
```javascript
new Date('2024-01-15 10:30:00')  // iOS返回Invalid Date
```

**原因**：
- iOS不支持 `YYYY-MM-DD HH:mm:ss` 格式

**解决方案**：

```typescript
/**
 * 安全的日期解析
 */
const safeDateParse = (dateStr: string): Date => {
  // 方案1：替换分隔符
  const safeStr = dateStr.replace(/-/g, '/');
  return new Date(safeStr);
};

/**
 * 方案2：使用dayjs（推荐）
 */
import dayjs from 'dayjs';

const formatDate = (dateStr: string) => {
  return dayjs(dateStr).format('YYYY-MM-DD HH:mm:ss');
};
```

## 5. 样式问题

### 5.1 问题：1px边框变粗

**现象**：
- 在高分辨率设备上，1px边框看起来很粗

**解决方案**：

```css
/**
 * 方案1：使用伪元素
 */
.thin-border {
  position: relative;
}

.thin-border::after {
  content: '';
  position: absolute;
  left: 0;
  bottom: 0;
  width: 100%;
  height: 1px;
  background: #ddd;
  transform: scaleY(0.5);
  transform-origin: 0 0;
}

/**
 * 方案2：使用viewport
 */
/* index.html */
<meta name="viewport" content="width=device-width, initial-scale=0.5, maximum-scale=0.5, user-scalable=no" />
/* 然后所有CSS像素乘以2 */
```

## 6. 调试技巧

### 6.1 Chrome DevTools 调试移动端

**步骤**：

1. **连接手机**
   ```
   Android手机 → 开启开发者模式 → 启用USB调试 → 连接电脑
   ```

2. **Chrome中调试**
   ```
   打开Chrome → 输入 chrome://inspect → 选择设备 → Inspect
   ```

3. **常用命令**
   ```javascript
   // 查看视口尺寸
   console.log(window.innerWidth, window.innerHeight);
   
   // 查看设备像素比
   console.log(window.devicePixelRatio);
   
   // 查看User Agent
   console.log(navigator.userAgent);
   ```
