## 三、错误处理体系

### 3.1 错误类型层次

```
APIError（基类）
├── AuthenticationError（认证错误）
├── RateLimitError（限流错误）
├── QuotaExceededError（配额超限）
├── ValidationError（参数验证错误）
├── NotFoundError（资源不存在）
├── ServerError（服务器错误）
├── NetworkError（网络错误）
├── TimeoutError（超时错误）
└── RetryExhaustedError（重试耗尽）
```

### 3.2 错误码分类

| 类别 | 错误码范围 | 说明 |
|------|------------|------|
| 成功 | 0 | 请求成功 |
| 参数错误 | 40001-400xx | 请求参数问题 |
| 认证错误 | 40101-401xx | 认证相关问题 |
| 权限错误 | 40301-403xx | 权限相关问题 |
| 资源错误 | 40401-404xx | 资源不存在 |
| 限流错误 | 42901-429xx | 频率/配额限制 |
| 服务错误 | 5xxxx | 服务器问题 |

## 五、业务代码模板

### 5.1 心理问答应用

```python
from api_platform import Client
from api_platform.exceptions import RateLimitError, QuotaExceededError

class PsychologyApp:
    def __init__(self, api_key):
        self.client = Client(api_key=api_key)
    
    def ask(self, question, user_id=None):
        try:
            response = self.client.psychology.chat(
                message=question,
                user_id=user_id
            )
            return {
                "success": True,
                "answer": response.answer,
                "suggestions": response.suggestions,
                "cost": response.usage.cost
            }
        except RateLimitError:
            return {"success": False, "error": "rate_limited"}
        except QuotaExceededError:
            return {"success": False, "error": "quota_exceeded"}
        except Exception as e:
            return {"success": False, "error": str(e)}
```

### 5.2 带重试的API调用

```python
import time

def call_with_retry(func, max_retries=5):
    for attempt in range(max_retries + 1):
        try:
            return func()
        except RateLimitError as e:
            wait_time = e.retry_after or (2 ** attempt)
            time.sleep(wait_time)
```
