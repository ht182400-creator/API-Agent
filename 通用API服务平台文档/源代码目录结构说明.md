## 核心模块说明

### 1. config - 配置模块
- `settings.py`: 应用程序配置，从环境变量加载
- `database.py`: 数据库连接配置，支持同步/异步

### 2. api - API路由
- `v1/auth.py`: 认证相关接口（注册、登录、Token刷新、签名生成）
- `v1/repositories.py`: 仓库列表、详情、调用接口
- `v1/quota.py`: 配额查询接口
- `v1/logs.py`: 调用日志查询接口

### 3. models - 数据模型
- `user.py`: 用户表、用户配置表
- `api_key.py`: API Key表、使用记录表
- `repository.py`: 仓库表、配置表、定价表、统计表
- `billing.py`: 账户表、账单表、配额表
- `adapter.py`: 适配器表、适配器实例表

### 4. services - 业务逻辑层
- `auth_service.py`: API Key验证、HMAC签名验证、Key管理
- `repo_service.py`: 仓库CRUD、审核、上下线
- `billing_service.py`: 充值、扣费、成本计算、退款
- `quota_service.py`: 限流检查、配额管理

### 5. adapters - 适配器层
- `base.py`: 适配器基类，定义标准接口
- `http_adapter.py`: HTTP适配器，用于REST API仓库
- `grpc_adapter.py`: gRPC适配器（预留）

### 6. core - 核心模块
- `exceptions.py`: 自定义异常类
- `security.py`: 密码哈希、JWT Token、HMAC签名
- `middleware.py`: 请求日志、限流、CORS中间件

## 快速启动

### 1. 安装依赖
```bash
cd api-platform
pip install -r requirements.txt
```

### 2. 配置环境变量
```bash
cp config/.env.example config/.env
# 编辑 config/.env 设置数据库密码等
```

### 3. 启动Docker服务
```bash
cd docker
docker-compose up -d
```

### 4. 初始化数据库
```bash
python scripts/init_db.py
python scripts/seed_data.py
```

### 5. 启动服务
```bash
# 开发模式（热重载）
uvicorn src.main:app --reload --host 0.0.0.0 --port 8080

# 或使用Makefile
make run
```

### 6. 访问API文档
- Swagger UI: http://localhost:8080/docs
- ReDoc: http://localhost:8080/redoc

## 运行测试

```bash
# 运行所有测试
pytest

# 运行测试并查看覆盖率
pytest --cov=src --cov-report=html

# 使用Makefile
make test
```

## 使用Makefile

```bash
make help          # 查看可用命令
make install       # 安装依赖
make dev           # 安装开发依赖
make test          # 运行测试
make lint          # 代码检查
make format        # 代码格式化
make docker-up     # 启动Docker服务
make docker-down   # 停止Docker服务
make db-init       # 初始化数据库
make db-seed       # 填充测试数据
```

- **框架**: FastAPI 0.109+
- **数据库**: PostgreSQL 15+, SQLAlchemy 2.0+

- **验证**: Pydantic v2
- **测试**: pytest
- **部署**: Docker, Docker Compose
