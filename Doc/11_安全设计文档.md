## 第二章 身份认证安全

### 2.1 密码安全

**密码策略**：
| 要求 | 规则 |
|------|------|
| 最小长度 | 8位 |
| 最大长度 | 32位 |
| 复杂度 | 必须包含大小写字母和数字 |
| 历史记录 | 不能重复使用最近5次密码 |
| 过期时间 | 90天（可选配置） |

**密码存储**：
```python
# 使用bcrypt加密
import bcrypt

def hash_password(password: str) -> str:
    """密码哈希"""
    salt = bcrypt.gensalt(rounds=12)
    hashed = bcrypt.hashpw(password.encode('utf-8'), salt)
    return hashed.decode('utf-8')

def verify_password(password: str, hashed: str) -> bool:
    """密码验证"""
    return bcrypt.checkpw(password.encode('utf-8'), hashed.encode('utf-8'))
```

### 2.2 JWT Token安全

**Token配置**：
| 类型 | 有效期 | 存储位置 |
|------|--------|----------|
| Access Token | 1小时 | 内存/JavaScript变量 |
| Refresh Token | 7天 | HttpOnly Cookie |

**Token生成**：
```python
from datetime import datetime, timedelta
import jwt

SECRET_KEY = os.getenv('JWT_SECRET_KEY')
ALGORITHM = 'HS256'

def create_access_token(user_id: str) -> str:
    """生成访问令牌"""
    expire = datetime.utcnow() + timedelta(hours=1)
    payload = {
        'sub': user_id,
        'exp': expire,
        'iat': datetime.utcnow(),
        'type': 'access'
    }
    return jwt.encode(payload, SECRET_KEY, algorithm=ALGORITHM)

def create_refresh_token(user_id: str) -> str:
    """生成刷新令牌"""
    expire = datetime.utcnow() + timedelta(days=7)
    payload = {
        'sub': user_id,
        'exp': expire,
        'iat': datetime.utcnow(),
        'type': 'refresh'
    }
    return jwt.encode(payload, SECRET_KEY, algorithm=ALGORITHM)
```

### 2.3 API Key安全

**Key生成**：
```python
import secrets
import hashlib

def generate_api_key() -> tuple:
    """生成API Key和Secret"""
    key = f"APIK_{secrets.token_urlsafe(27)}"
    secret = f"sk_{secrets.token_urlsafe(27)}"
    secret_hash = hashlib.sha256(secret.encode()).hexdigest()
    return key, secret, secret_hash
```

**签名验证**：
```python
import hmac
import hashlib
import time

def verify_signature(secret_hash: str, signature: str, method: str, 
                    path: str, timestamp: str, body: str) -> bool:
    """验证请求签名"""
    # 检查时间戳是否在5分钟内
    if abs(int(timestamp) - int(time.time())) > 300:
        return False
    
    # 计算期望签名
    body_hash = hashlib.sha256(body.encode()).hexdigest()
    string_to_sign = f"{method}\n{path}\n{timestamp}\n{body_hash}"
    expected = hmac.new(
        secret_hash.encode(),
        string_to_sign.encode(),
        hashlib.sha256
    ).hexdigest()
    
    return hmac.compare_digest(signature, expected)
```

## 第四章 API安全防护

### 4.1 限流策略

**多维度限流**：
| 维度 | 默认值 | 说明 |
|------|--------|------|
| IP限流 | 1000次/小时 | 防止DDoS |
| 用户限流 | 10000次/天 | 防止滥用 |
| Key限流 | 100次/秒 | 资源控制 |
| 租户限流 | 按套餐 | SLA保障 |

**限流实现**：
```python
import redis
from fastapi import Request, HTTPException

redis_client = redis.Redis(host='redis', decode_responses=True)

async def rate_limit(key: str, limit: int, window: int) -> bool:
    """滑动窗口限流"""
    current = redis_client.incr(key)
    if current == 1:
        redis_client.expire(key, window)
    if current > limit:
        raise HTTPException(status_code=429, detail="Rate limit exceeded")
    return True

# 使用示例
@app.middleware("api_key")
async def check_rate_limit(request: Request, call_next):
    api_key = request.headers.get('X-API-Key')
    
    # IP限流
    ip = request.client.host
    await rate_limit(f"rate:ip:{ip}", limit=1000, window=3600)
    
    # Key限流
    await rate_limit(f"rate:key:{api_key}", limit=100, window=1)
    
    response = await call_next(request)
    return response
```

### 4.2 输入验证

**参数校验**：
```python
from pydantic import BaseModel, EmailStr, validator, constr
import re

class UserRegister(BaseModel):
    email: EmailStr
    password: constr(min_length=8, max_length=32)
    name: constr(min_length=1, max_length=100)
    
    @validator('password')
    def validate_password(cls, v):
        if not re.match(r'^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)', v):
            raise ValueError('密码必须包含大小写字母和数字')
        return v

class ApiKeyCreate(BaseModel):
    name: constr(min_length=1, max_length=100)
    scopes: list[str] | None = None
    
    @validator('scopes')
    def validate_scopes(cls, v):
        allowed = {'read', 'write', 'admin'}
        if v and not all(s in allowed for s in v):
            raise ValueError(f'无效的权限范围: {v}')
        return v
```

### 4.4 XSS防护

**防护措施**：
```python
import html
from fastapi import Request, Response

@app.middleware("http")
async def xss_protection(request: Request, call_next):
    response = await call_next(request)
    
    # 添加安全响应头
    response.headers['X-Content-Type-Options'] = 'nosniff'
    response.headers['X-XSS-Protection'] = '1; mode=block'
    response.headers['Content-Security-Policy'] = "default-src 'self'"
    
    return response

def escape_html(text: str) -> str:
    """HTML转义"""
    return html.escape(text)
```

## 第五章 数据安全

### 5.1 敏感数据处理

**敏感字段**：
| 字段类型 | 处理方式 |
|----------|----------|
| 密码 | bcrypt加密 |
| 手机号 | 部分掩码：138****5678 |
| 身份证 | 部分掩码：310***********1234 |
| 银行卡 | 部分掩码：**** **** **** 1234 |
| 邮箱 | 部分掩码：a***@example.com |

**脱敏函数**：
```python
import re

def mask_phone(phone: str) -> str:
    """手机号脱敏"""
    if len(phone) == 11:
        return f"{phone[:3]}****{phone[-4:]}"
    return phone

def mask_email(email: str) -> str:
    """邮箱脱敏"""
    parts = email.split('@')
    if len(parts) == 2:
        name = parts[0]
        domain = parts[1]
        if len(name) > 1:
            return f"{name[0]}***@{domain}"
    return email

def mask_id_card(id_card: str) -> str:
    """身份证脱敏"""
    if len(id_card) >= 15:
        return f"{id_card[:6]}********{id_card[-4:]}"
    return id_card
```

### 5.2 传输加密

**HTTPS配置**：
```nginx
# Nginx HTTPS配置
server {
    listen 443 ssl http2;
    server_name api.example.com;

    ssl_certificate /etc/nginx/ssl/cert.pem;
    ssl_certificate_key /etc/nginx/ssl/key.pem;

    # TLS版本
    ssl_protocols TLSv1.2 TLSv1.3;

    # 加密套件
    ssl_ciphers ECDHE-ECDSA-AES128-GCM-SHA256:ECDHE-RSA-AES128-GCM-SHA256;
    ssl_prefer_server_ciphers on;

    # HSTS
    add_header Strict-Transport-Security "max-age=31536000; includeSubDomains" always;

    # OCSP Stapling
    ssl_stapling on;
    ssl_stapling_verify on;
}
```

### 5.3 数据加密存储

**字段加密**：
```python
from cryptography.fernet import Fernet
from sqlalchemy import Column, String

class EncryptedString(TypeDecorator):
    impl = String
    cache_encryption = None

    def __init__(self, key=None, **kwargs):
        self.cipher = Fernet(key or settings.ENCRYPTION_KEY)
        super().__init__(**kwargs)

    def process_bind_param(self, value, dialect):
        if value is not None:
            return self.cipher.encrypt(value.encode()).decode()

    def process_result_value(self, value, dialect):
        if value is not None:
            return self.cipher.decrypt(value.encode()).decode()

class SensitiveData(Model):
    api_secret = EncryptedString()
```
