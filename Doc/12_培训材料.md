## 第五章 核心模块培训

### 5.1 认证模块

**JWT认证流程**：
```
用户登录 → 验证密码 → 生成Token → 返回Token
    │
    ▼
后续请求 → 提取Token → 验证签名 → 验证过期 → 提取用户ID → 请求处理
```

**代码示例**：
```python
from fastapi import Depends, HTTPException
from fastapi.security import HTTPBearer, HTTPAuthorizationCredentials

security = HTTPBearer()

async def get_current_user(
    credentials: HTTPAuthorizationCredentials = Depends(security)
) -> User:
    """获取当前用户"""
    token = credentials.credentials
    
    try:
        payload = jwt.decode(token, SECRET_KEY, algorithms=[ALGORITHM])
        user_id = payload.get("sub")
        if user_id is None:
            raise HTTPException(status_code=401, detail="无效的Token")
    except JWTError:
        raise HTTPException(status_code=401, detail="Token已过期")
    
    user = get_user_by_id(user_id)
    if user is None:
        raise HTTPException(status_code=401, detail="用户不存在")
    
    return user
```

### 5.2 API Key模块

**Key生成流程**：
```
申请Key → 生成Key/Secret → 存储Hash → 返回明文Secret（仅一次）
    │
    ▼
使用Key → 提取Key → 查询验证 → 检查状态 → 检查配额 → 请求处理
```

**代码示例**：
```python
@router.post("/keys", response_model=APIKeyResponse)
async def create_api_key(
    current_user: User = Depends(get_current_user)
):
    """创建API Key"""
    # 生成Key和Secret
    key = f"APIK_{secrets.token_urlsafe(27)}"
    secret = f"sk_{secrets.token_urlsafe(27)}"
    secret_hash = hashlib.sha256(secret.encode()).hexdigest()
    
    # 存储到数据库
    api_key = APIKey(
        user_id=current_user.id,
        api_key=key,
        secret_hash=secret_hash,
        name="默认Key"
    )
    db.add(api_key)
    db.commit()
    
    # 返回明文Secret（仅此次）
    return {
        "api_key": key,
        "secret": secret,  # 仅在此处返回
        "name": "默认Key"
    }
```

### 5.3 限流模块

**限流算法**：滑动窗口算法

```
窗口大小: 1秒
最大请求: 100次

┌─────────────────────────────────────┐
│  时间轴 ─────────────────────────▶  │
│  ├────┤                          │
│  0s   现在                        │
│       ↑                           │
│       当前请求计入此窗口           │
└─────────────────────────────────────┘
```

**代码示例**：
```python
from fastapi import Request, HTTPException
import redis

redis_client = redis.Redis(host='redis')

async def rate_limit(request: Request):
    """限流检查"""
    api_key = request.headers.get('X-API-Key')
    key = f"ratelimit:{api_key}"
    
    # 使用Redis INCR和EXPIRE实现滑动窗口
    current = redis_client.incr(key)
    if current == 1:
        redis_client.expire(key, 1)
    
    if current > 100:  # 100 QPS
        raise HTTPException(
            status_code=429,
            detail="请求过于频繁",
            headers={"Retry-After": "1"}
        )
```

## 第七章 安全规范培训

### 7.1 密码安全

**密码要求**：
- 长度至少8位
- 必须包含大小写字母和数字
- 不要使用弱密码（如123456、password等）
- 定期更换密码

### 7.2 API Key安全

**Key使用规范**：
- Secret只在创建时返回，请立即保存
- 不要在客户端代码中硬编码Key
- 定期轮换Key
- 发现泄露立即禁用

**签名验证**：
```python
import hmac
import hashlib
import time

def generate_signature(secret: str, method: str, path: str, body: str) -> str:
    timestamp = str(int(time.time()))
    body_hash = hashlib.sha256(body.encode()).hexdigest()
    string_to_sign = f"{method}\n{path}\n{timestamp}\n{body_hash}"
    signature = hmac.new(secret.encode(), string_to_sign.encode(), hashlib.sha256).hexdigest()
    return signature
```

### 7.3 数据安全

**敏感数据处理**：
- 密码必须加密存储
- 日志中禁止记录敏感信息
- 敏感数据查询结果需脱敏
