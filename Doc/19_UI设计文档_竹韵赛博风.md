## 1. 设计概述

### 1.1 设计理念

API 管理平台采用**翠竹主题**设计理念，融合：
- **竹韵元素**：翠竹、竹节、竹叶等中国传统意象
- **现代风格**：现代科技感的卡片、按钮、渐变效果
- **明快色调**：清新明亮的翡翠绿配色，舒适护眼

### 1.2 版本历史

| 版本 | 日期 | 更新内容 |
|------|------|----------|
| V1.0 | 2026-04-16 | 初始深色主题 |
| V2.0 | 2026-04-24 | 明快版，浅色背景 |
| V2.1 | 2026-04-24 | 翠竹版，翠竹主题、绿色基调 |

## 2. 色彩系统

### 2.1 主色调

```css
:root {
  /* 翡翠绿系 - 主色调 */
  --emerald: #059669;           /* 翡翠绿 - 主色 */
  --emerald-light: #10b981;     /* 亮翡翠 */
  --emerald-dark: #047857;      /* 深翡翠 */
  --emerald-bg: #ecfdf5;        /* 翡翠浅底 */

  /* 朱红系 - 点缀色 */
  --cinnabar: #dc2626;          /* 朱红 - 强调 */
  --cinnabar-light: #ef4444;    /* 亮朱红 */
  --gold: #b45309;              /* 琥珀金 */
  --azure: #2563eb;            /* 天蓝 */

  /* 中性色 - 文字 */
  --ink-black: #f8fafc;         /* 墨白 */
  --ink-deep: #f1f5f9;         /* 墨灰 */
  --ink-surface: #ffffff;       /* 墨面 */
  --ink-elevated: #e2e8f0;     /* 墨凸 */

  /* 文字色 */
  --text-primary: #1e293b;      /* 主文字 */
  --text-secondary: #64748b;    /* 次文字 */
  --text-muted: #94a3b8;       /* 弱文字 */
}
```


## 4. 印章装饰系统

### 4.1 印章设计

印章位于页面右下角，作为竹韵主题的标志性装饰：

```tsx
<div className="china-stamp">
  <svg viewBox="0 0 60 60" xmlns="http://www.w3.org/2000/svg">
    {/* 外框 */}
    <rect x="5" y="5" width="50" height="50" 
          fill="none" stroke="#059669" strokeWidth="2" rx="4"/>
    {/* 内框 */}
    <rect x="10" y="10" width="40" height="40" 
          fill="none" stroke="#059669" strokeWidth="0.5"/>
    {/* 主文字 */}
    <text x="30" y="35" textAnchor="middle" fontSize="16" 
          fill="#059669" fontFamily="serif" fontWeight="bold">API</text>
    {/* 副文字 */}
    <text x="30" y="48" textAnchor="middle" fontSize="8" 
          fill="#dc2626" fontFamily="serif">墨韵</text>
  </svg>
</div>
```

### 4.2 印章样式

```css
.china-stamp {
  position: fixed;
  bottom: 20px;
  right: 20px;
  width: 70px;
  height: 70px;
  opacity: 0.5;

  z-index: 1;
  filter: drop-shadow(0 2px 4px rgba(5, 150, 105, 0.2));
}

.china-stamp svg {
  width: 100%;
  height: 100%;
}
```

## 7. 使用指南

### 7.1 启用竹韵主题

在页面根元素添加 `bamboo-bg-pattern` 类：

```tsx
import styles from './Dashboard.module.css';

function Dashboard() {
  return (
    <div className={`${styles.dashboard} bamboo-bg-pattern`}>
      {/* 页面内容 */}
    </div>
  );
}
```

### 7.2 自定义颜色

通过覆盖 CSS 变量修改主题色：

```css
:root {
  --emerald: #0d9488;      /* 自定义主色：青色 */
  --bg-primary: linear-gradient(160deg, #f0fdfa 0%, #ccfbf1 100%);
}
```

### 7.3 调整装饰透明度

```css
/* 降低竹林纹理透明度 */
.bamboo-bg-pattern::before {
  opacity: 0.4;
}

/* 降低角落光晕透明度 */
.bamboo-bg-pattern::after {
  opacity: 0.5;
}
```
