使用帮助

          
        

        

          

            
1. 获取 API Key

            
登录 Developer Portal，进入「API Key 管理」页面创建或复制 Key。

          

          

            
2. 配置测试工具

            
在「API 配置」区域输入 API Key 并保存。

          

          

            
3. 发送请求

            
选择接口类型，填写参数，点击「发送请求」。

          

          

            
快捷键

            
    
              
  • Ctrl + Enter：发送请求
    • 
              
  • Ctrl + C（响应区）：复制响应内容
