## 1. 版本概述

Weather API 调用测试工具 v1.0.0 是面向开发者的轻量级 Web 工具，用于快速测试和验证 API 平台的天气服务接口。

### 1.1 核心价值

- **零配置**：无需安装，浏览器直接使用
- **快速验证**：3 步完成 API 调用测试
- **清晰展示**：格式化 JSON 响应，易于阅读

## 4. 使用示例

### 4.1 快速开始

**Step 1**: 获取 API Key
1. 登录 Developer Portal
2. 进入「API Key 管理」页面
3. 创建或复制现有 Key

**Step 2**: 配置测试工具
1. 打开测试工具页面
2. 粘贴 API Key
3. 点击「保存」

**Step 3**: 调用接口
1. 选择接口类型
2. 输入城市名称
3. 点击「发送请求」

### 4.2 调用示例

```bash
# cURL 示例
curl -X GET "http://localhost:9000/api/v1/weather/current?city=北京" \
  -H "Authorization: Bearer YOUR_API_KEY"

# JavaScript 示例
const response = await fetch(
  'http://localhost:9000/api/v1/weather/current?city=北京',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);
const data = await response.json();
console.log(data);
```
